В контексте исходного кода [class] предоставь переработанную версию фрагмента кода [snippet].

Если код не требует рефакторинга - ответь сообщением "Код не требует рефакторинга".

Объяснение должно быть кратким и компактным.

Не печатай содержимое всего класса. Только измененный фрагмент кода и методы, которые в нем используются.

Ответ должын быть в формате markdown и соответствовать шаблону:

<твое объяснение и рассуждения по рефакторингу>

```<Язык программирования>
<Твой код>
```

Давай разберемся с этим шаг за шагом, чтобы быть уверенными, что у нас есть правильный ответ.

[snippet]=
```${lang}
${selectedText}
```


[class]=
```${lang}
${documentText}
```
